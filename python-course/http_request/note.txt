Metodos HTTP
    GET : servidor → Cliente
    POST : Guardar Cliente → Servidor
    PUT: modificaion de datos Cliente → Servidor
    Delete: Eliminar Datos Cliente → Servidor

Codigos de estado HTTP:
1xx →   respeustas informativas
2xx →   Respuestas satifactores
3xx →   redireccion
4xx →   error Cliente
5xx →   Error de servidor
